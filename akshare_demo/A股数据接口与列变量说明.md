# Akshare 能拉到的 A 股数据概览（以 1～2 支股票为例）

基于 akshare 源码整理，**行情类**主要用 2 个接口即可覆盖日线 + 分钟线。下面列出各接口的**周期**、**列变量**和**说明**。

---

## 一、日线 / 周线 / 月线（同一接口，不同周期）

**接口**: `ak.stock_zh_a_hist(symbol, period=..., start_date, end_date, adjust=...)`

| 参数 | 说明 |
|------|------|
| `symbol` | 6 位股票代码，如 `"000001"`（平安银行）、`"600519"`（贵州茅台） |
| `period` | `"daily"` 日线 / `"weekly"` 周线 / `"monthly"` 月线 |
| `start_date` / `end_date` | 格式 `"YYYYMMDD"` |
| `adjust` | `""` 不复权 / `"qfq"` 前复权 / `"hfq"` 后复权 |

**返回列变量（共 12 列）**:

| 列名   | 含义     |
|--------|----------|
| 日期   | 交易日期 |
| 股票代码 | 标的代码 |
| 开盘   | 开盘价   |
| 收盘   | 收盘价   |
| 最高   | 最高价   |
| 最低   | 最低价   |
| 成交量 | 成交量（股） |
| 成交额 | 成交金额（元） |
| 振幅   | 振幅     |
| 涨跌幅 | 涨跌幅（%） |
| 涨跌额 | 涨跌额   |
| 换手率 | 换手率（%） |

---

## 二、分钟线（1 / 5 / 15 / 30 / 60 分钟）

Akshare 里 A 股分钟线有 **2 个数据源**：

| 数据源 | 接口 | 1 分钟 | 5/15/30/60 分钟 | 说明 |
|--------|------|--------|------------------|------|
| **东方财富** | `stock_zh_a_hist_min_em` | 有，但易断连 | ✅ 稳定 | 可指定起止日期；1 分钟走 trends2 接口，易被关连接 |
| **新浪** | `stock_zh_a_minute` | ✅ 稳定 | ✅ 支持 | 不能指定日期，返回最近约 1970 根 K 线；symbol 需带 `sh`/`sz` 前缀 |

### 东方财富：`ak.stock_zh_a_hist_min_em(symbol, start_date, end_date, period=..., adjust=...)`

| 参数 | 说明 |
|------|------|
| `symbol` | 6 位股票代码 |
| `start_date` / `end_date` | 格式 `"YYYY-MM-DD HH:MM:SS"`，如 `"2024-03-01 09:30:00"` |
| `period` | `"1"` / `"5"` / `"15"` / `"30"` / `"60"` 分钟 |
| `adjust` | `""` / `"qfq"` / `"hfq"` |

### 1 分钟线返回列（8 列）

| 列名 | 含义     |
|------|----------|
| 时间 | 时间戳   |
| 开盘 | 开盘价   |
| 收盘 | 收盘价   |
| 最高 | 最高价   |
| 最低 | 最低价   |
| 成交量 | 成交量   |
| 成交额 | 成交额   |
| 均价 | 均价     |

### 5 / 15 / 30 / 60 分钟线返回列（10 列）

| 列名   | 含义     |
|--------|----------|
| 时间   | 时间戳   |
| 开盘   | 开盘价   |
| 收盘   | 收盘价   |
| 最高   | 最高价   |
| 最低   | 最低价   |
| 涨跌幅 | 涨跌幅   |
| 涨跌额 | 涨跌额   |
| 成交量 | 成交量   |
| 成交额 | 成交额   |
| 振幅   | 振幅     |
| 换手率 | 换手率   |

### 新浪（推荐 1 分钟用）：`ak.stock_zh_a_minute(symbol, period=..., adjust=...)`

| 参数 | 说明 |
|------|------|
| `symbol` | **带前缀**：`sh600519`、`sz000001` |
| `period` | `"1"` / `"5"` / `"15"` / `"30"` / `"60"` 分钟 |
| `adjust` | `""` 不复权 / `"qfq"` 前复权 / `"hfq"` 后复权 |

- 不支持起止日期，返回**最近约 1970 根**分钟 K 线（约 8～13 个交易日）。
- **返回列（7 列）**：`day`, `open`, `high`, `low`, `close`, `volume`, `amount`（与 Qlib 常用列名接近，便于映射）。

## 三、其他常见 A 股接口（列变量较多，仅列名称）

- **全市场实时行情** `stock_zh_a_spot_em()`：最新价、涨跌幅、成交量、成交额、市盈率、市净率、总市值、流通市值、量比、换手率等很多列。
- **单只个股实时**：可用 `stock_zh_a_spot_em()` 再按代码筛选，或使用其它单只接口（若有）。
- **财务/基本面**：如 `stock_financial_abstract(stock=...)`、`stock_financial_analysis_indicator(symbol=...)` 等，列非常多（报表项目、指标等），需要时查官方文档即可。

---

## 四、列变量汇总（仅行情）

| 数据类型     | 接口 | 列数 | 核心列 |
|--------------|------|------|--------|
| 日/周/月线   | `stock_zh_a_hist` | 12 | 日期、开高低收、成交量、成交额、涨跌幅、涨跌额、换手率、振幅、股票代码 |
| 1 分钟线（东财） | `stock_zh_a_hist_min_em(period="1")` | 8  | 时间、开高低收、成交量、成交额、均价（易断连） |
| **1 分钟线（新浪）** | `stock_zh_a_minute(symbol="sz000001", period="1")` | **7** | **day, open, high, low, close, volume, amount**（稳定，含 11 号等近期日） |
| 5/15/30/60 分钟 | `stock_zh_a_hist_min_em(period="5" 等)` | 10 | 时间、开高低收、涨跌幅、涨跌额、成交量、成交额、振幅、换手率 |

用 1～2 支股票（如 `000001`、`600519`）分别调上述接口，即可看到实际返回的列与上面一致；若网络不通可先看本文档了解“能拉哪些列”。
